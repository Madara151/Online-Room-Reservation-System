Quick Guide for Staff

Follow these steps to use the system smoothly.

1) Login

Open the system and login using your username and password. If you are testing, you can register a staff user from the login page.

2) Add Reservation

Go to Reservations page, fill guest details, select room type, enter dates in YYYY-MM-DD format, and click Save Reservation.

3) View Reservation

In Reservations page, use Find Reservation and search by Reservation No (e.g., R001).

4) Generate Bill

Go to Billing page, enter Reservation No and click Generate. You can print using Print.

5) Reports

Go to Reports page, enter From/To date range and load bookings or revenue.

Common issues:
• If you get 401 error → your login token expired → login again.
• Date format must be YYYY-MM-DD.
